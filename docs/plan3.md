# Plan Sprint 3: Interactividad y Narrativa Visual
**Objetivo:** Implementar animaciones de entrada y mejorar la experiencia de usuario (UX).

**Tareas:**
- Instalación e integración de GSAP.
- Creación de Timeline de carga para la sección Hero.
- Optimización de la fluidez visual.

**Metodología:**
- Uso de GSAP para evitar el uso excesivo de animaciones CSS pesadas.
- Enfoque en "Performance" para asegurar que las animaciones no afecten al tiempo de carga (Lighthouse score).
